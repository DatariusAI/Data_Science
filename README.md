## Overview

This repository is a portfolio of data science projects showcasing the full analytical pipeline. From raw data ingestion and cleaning through exploratory data analysis, statistical testing, and visualization -- each project demonstrates practical, real-world data science skills.

The focus is on clear methodology, reproducible analysis, and communicating findings effectively through visualizations and narrative.

## Getting Started

### Prerequisites

```bash
pip install numpy pandas matplotlib seaborn plotly scipy jupyter
```

### Running Notebooks

```bash
jupyter notebook
```

Navigate to the project folder of interest and open the corresponding `.ipynb` file.
